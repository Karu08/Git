TASK2

This folder contains tasks designed to understand the usage of .gitignore file to exclude certain files or directories.

Objectives:

- Initialize a new Git repository.
- Create files within the repository.
- Create and Add .gitignore file with patterns.
- Add files that do not match this pattern.
- Use git status to verify the files that are being tracked.


Prerequisites:

Before starting, ensure that the following are installed in the system:
- Git
- Text Editor or IDE


Files in this folder:
1) Task2.pdf - contains the implementation of Task2
2) sample1.txt, testfile.log, after.log - files that have been used for verifying the usage of .gitignore
